Terms of Service Privacy Policy

1 Data We Collect

SanchoHelper stores only configuration and operational data per Discord server:

  • Server ID — To identify your server
  • Role IDs — Owner role, staff roles, marked role
  • Channel IDs — Log, welcome, and announcement channels
  • Custom Messages — Welcome text you provide
  • Appeal Records — User ID, reason, timestamp, status

2 What We NEVER Collect

  • Message content
  • Email addresses
  • IP addresses
  • Passwords or tokens
  • Any personal identifying information beyond Discord IDs

3 How Data Is Used

All data is used exclusively for Bot functionality:

  • Server ID → Load correct settings
  • Role IDs → Alert and ping correct staff
  • Channel IDs → Send logs to correct places
  • Appeal records → Process restriction appeals

4 Data Storage

Data is stored in JSON files on the hosting server. We take reasonable precautions but cannot guarantee 100% security of any data storage method.

5 Data Sharing

We do NOT sell, trade, rent, or share your data with any third parties. Ever.

6 Data Removal

To request deletion of your server's data:

  • Kick the Bot from your server (data becomes inactive)
  • Contact the creator for full deletion: Discord ID 1192124461281976361

7 Children's Privacy

The Bot is not directed at children under 13. By using the Bot, you confirm you meet Discord's minimum age requirement (13+).

8 Changes

This policy may be updated. Major changes will be announced via the Bot's update system. Continued use means acceptance.
